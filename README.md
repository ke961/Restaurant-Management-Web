## ✨ Features

| Feature | Description |
|---|---|
| 🏠 **Landing Page** | Beautiful, modern hero section to welcome customers |
| 🍔 **Interactive Menu** | Browse all food items with categories and details |
| 🛒 **Cart System** | Add, remove, and adjust quantities seamlessly |
| 💰 **Auto Billing** | Real-time price calculation as you build your order |
| ✅ **Order Confirmation** | Smooth order placement with confirmation feedback |
| 📜 **Order History** | View all past orders in one place |
| 📱 **Responsive Design** | Fully optimized for mobile, tablet, and desktop |
| ⚡ **Blazing Fast** | Powered by Vite for instant HMR and fast builds |

## 📦 Installation & Setup

> **Prerequisites:** Node.js ≥ 16 and npm installed on your machine.

**1. Clone the repository**
```bash
git clone https://github.com/ke961/Restaurant-Management-Web.git
```

**2. Navigate to the frontend**
```bash
cd "Restaurant Management Web/Frontend"
```

**3. Install dependencies**
```bash
npm install
```

**4. Start the development server**
```bash
npm run dev
```

Then open **[http://localhost:5173](http://localhost:5173)** in your browser. 🚀
